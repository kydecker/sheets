# Sheets

This respository is a collection of sheet music that I ([Ky Decker](https://ky.fyi)) arranged and typeset for solo piano.

Sheets are written using [LilyPond](https://lilypond.org/), a text-based music engraving language.

## Setup

Install LilyPond command line tools:

```bash
brew install lilypond
```

Open the `.ly` and `.pdf` files side-by-side in VSCode. Make sure that the PDF preview is open with "Lilypond PDF Preview" from the recommended [VSLilyPond](https://marketplace.visualstudio.com/items?itemName=lhl2617.vslilypond) extension.

Each time the `.ly` file is saved, the `.pdf` will be refreshed. Changes should display automatically.
